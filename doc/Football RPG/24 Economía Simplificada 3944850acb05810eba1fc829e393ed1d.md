# 24. Economía Simplificada

# Modelo de franquicia igualitario

**Presupuesto igual para todas las franquicias.** No hay gestión financiera, no hay déficit, no hay quiebra. La competencia es puramente deportiva y de deckbuilding.

**Todos los jugadores cobran igual.** Sin negociaciones salariales, sin estructura salarial. Elimina la gestión tediosa de contratos y salarios.

**El presupuesto se usa para:**

- Contratar staff (4 roles, cada uno con coste según nivel de estrellas)
- Mejorar instalaciones de cantera (5 niveles, inversión a largo plazo)
- Ambos compiten por el mismo presupuesto fijo

**La tensión económica simplificada:** con presupuesto limitado igual para todos, la decisión es ¿invierto en mejor staff (beneficio inmediato) o en mejores instalaciones de cantera (beneficio a largo plazo)? Un club no puede tener TODO al máximo: debe priorizar.

**Sin contratos individuales.** Los jugadores se retienen o descartan al final de temporada (ver sección 23). No hay duración de contrato, no hay cláusulas de rescisión, no hay renovaciones. El ciclo de retención + draft sustituye todo el sistema contractual.
